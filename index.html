The pointer CSS media feature tests whether the user has a pointing device (such as a mouse), and if so, how accurate the primary pointing device is.

The pointer media query has three values:

  1. none - device has no pointer
  2. coarse - pointer has limited accuracy
  3. fine - pointer has high level of accuracy

Example

This example creates a small checkbox and radio button for users with fine primary pointers and a large checkbox for users with coarse primary pointers.

The form controls below will be enlarged for devices with a coarse pointer

      
        
        
      
    
      
        @media (pointer:coarse) {
          input[type="checkbox"], input[type="radio"] {
            min-width:30px;
            min-height:40px;
            background:transparent;
          }
        }

The any-pointer CSS media feature tests whether the user has any pointing device (such as a mouse), and if so, how accurate it is.

The any-pointer media query has three values:

  1. none - No pointing device is available.
  2. coarse - At least one input mechanism includes a pointing device of limited accuracy.
  3. fine - At least one input mechanism includes an accurate pointing device.

Example

This example creates a small checkbox and radio button for users with fine primary pointers and a large checkbox for users with coarse primary pointers.

The form controls below will be enlarged for devices with a coarse pointer

      
        
        
      
    
      
        @media (any-pointer:coarse) {
          input[type="checkbox"], input[type="radio"] {
            min-width:30px;
            min-height:40px;
            background:transparent;
          }
        }

The hover CSS media feature can be used to test whether the user's primary input mechanism can hover over elements.

The hover media query has two values:

  1. none - The primary input mechanism cannot hover at all or cannot conveniently hover (e.g., many mobile devices emulate hovering when the user performs an inconvenient long tap), or there is no primary pointing input mechanism.
  2. hover - The primary input mechanism can conveniently hover over elements.

Example

This example creates a small checkbox and radio button for users with fine primary pointers and a large checkbox for users with coarse, primary pointers.

The form controls below will be enlarged for devices with a coarse pointer

      
         Look at me!
      
    
      
        @media (hover: hover) {
          a:hover {
            background: yellow;
          }
